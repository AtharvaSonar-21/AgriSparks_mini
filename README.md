# Plant Disease Classification System

AI-powered plant disease detection using TensorFlow.js and MERN stack.

## Features
- Real-time disease detection
- 95%+ accuracy
- Mobile-friendly interface
- Prediction history
- RESTful API

## Setup

### Backend
\`\`\`bash
cd backend
npm install
# Add your MongoDB URI to .env
npm run dev
\`\`\`

### Frontend
\`\`\`bash
cd frontend
npm install
npm start
\`\`\`

## Tech Stack
- MongoDB
- Express.js
- React.js
- Node.js
- TensorFlow.js
