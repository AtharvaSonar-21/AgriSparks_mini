# Plant Disease Classification System

AI-powered plant disease detection using TensorFlow.js and MERN stack.

## Setup

### Backend
```bash
cd backend
npm install
# Add your MongoDB URI to .env
npm run dev
```

### Frontend
```bash
cd frontend
npm install
npm start
```

## Tech Stack
- MongoDB
- Express.js
- React.js
- Node.js
- TensorFlow.js
